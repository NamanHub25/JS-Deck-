# Alerts & Prompts 
Whenever we write code inside console.log() it gets printed on the console likewise if we wanna print errors on screen we use console.error() and for warning we use console.warn()

Alerts display an alert message on the page (doesn't ask for an input)
```js
alert("This is an alert message!")
```
Prompts displays a dialog box that on the page for the user input
```js
prompt("Please enter something!")
```
